Games Library Fullstack app using Vue.js, Bootstrap and Flask.

Backend:
1. Create .env using the example
2. Open first terminal
3. Type: 'cd backend'
4. Type: 'python -m venv venv' / 'python3 -m venv venv'
5. Type: Windows - 'venv/Scripts/activate' | Mac -source './venv/bin/activate'
6. Type: 'pip install -r ./requirements.txt'
7. Type: 'python main.py'

Frontend:
1. Create .env using the example
2. Open second terminal
3. Type: 'cd frontend'
4. Type: 'npm i'
5. Type: 'npm run serve'

* When installing additional packages use: 'pip freeze > ./requirements.txt' (make sure venv is activated).
